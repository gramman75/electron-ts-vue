# electron-ts-vue
electron-typescript-vue 사용을 위한 template
vue는 class base component를 사용함.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
